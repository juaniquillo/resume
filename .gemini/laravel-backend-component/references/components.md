# Component Enums

The `Juaniquillo\BackendComponents\Enums\ComponentEnum` (https://github.com/juaniquillo/laravel-backend-component) provides a type-safe way to reference available components.

## Available Components

### Layout & Containers
- `TEMPLATE`: 'template'
- `COLLECTION`: 'collection'
- `DIV`: 'div'
- `PARAGRAPH`: 'paragraph'

### Inline Elements
- `BUTTON`: 'inline.button'
- `LINK`: 'inline.link'
- `IMG`: 'inline.img'
- `SPAN`: 'inline.span'
- `BOLD`: 'inline.bold'
- `EM`: 'inline.em'
- `ITALIC`: 'inline.italic'
- `STRONG`: 'inline.strong'
- `SMALL`: 'inline.small'

### Headers
- `H1`: 'headers.h1'
- `H2`: 'headers.h2'
- ... `H6`

### Forms
- `FORM`: 'form.form'
- `LABEL`: 'form.label'
- `TEXT_INPUT`: 'form.text'
- `TEXTAREA`: 'form.textarea'
- `SELECT`: 'form.select'
- ... (Other standard HTML inputs)

### Tables
- `TABLE`: 'table.table'
- `THEAD`: 'table.thead'
- `TBODY`: 'table.tbody'
- `TR`: 'table.tr'
- `TH`: 'table.th'
- `TD`: 'table.td'

### Lists
- `OL`: 'lists.ol'
- `UL`: 'lists.ul'
- `LI`: 'lists.li'

### Custom
- `MODAL`: 'custom.modal'
